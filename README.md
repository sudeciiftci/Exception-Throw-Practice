# Java Exception & Throw Practice

This project demonstrates **exception handling** and the use of the **`throw` keyword** in Java.

## Concepts
- General exception handling
- Exception hierarchy
- `throw` keyword
- `IllegalArgumentException`
- `try-catch-finally`

## Examples

### exception1
- Shows why `Exception` must be caught **after** specific exceptions.
- Demonstrates exception hierarchy.

### throw1
- Throws an exception when age is negative.

### throw2
- Simulates a bank withdrawal.
- Validates withdrawal amount and balance.
- Uses `throw`, `try-catch`, and `finally`.

## Purpose
- Understand how to throw exceptions manually
- Learn exception handling order
- Practice input validation

## Run
Run `exception1`, `throw1`, and `throw2` separately.
